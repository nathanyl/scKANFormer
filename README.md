# scKANFormer

## Environment
- python >= 3.8
- PyTorch >= 2.0
- `numpy`
- `pandas`
- `scipy`
- `scikit-learn`
- `scanpy>=1.8.2`
- `anndata>=0.7.2`
- `matplotlib`
- `seaborn`
- `numba`
- `louvain`
- `umap-learn`
- `einops`
- `tqdm>=4.62.1`

## Installation
```
git clone https://github.com/nathanyl/scKANFormer
```

## Run scKANFormer
Run the training pipeline:
- python train.py
Run the inference (prediction) pipeline:
- python predict.py
